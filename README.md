# React Sneakers

Учебный интернет-магазин кроссовок на React.

## Что есть в проекте

- каталог кроссовок;
- поиск по товарам;
- избранное;
- корзина;
- оформление заказа;
- страница заказов;
- fallback-данные на случай недоступности MockAPI;
- деплой на GitHub Pages.

## Стек

- React
- React Router
- Axios
- SCSS Modules
- Macro CSS
- Create React App

## Запуск

npm install
npm start

## Сборка

npm run build

## Деплой

npm run deploy
